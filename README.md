# Organic Router

Organic is a lightweight, fast and flexible PHP router designed with modern PSR standards.

The goal of Organic is to provide a simple and powerful routing system that supports:

* High performance routing
* Subdomain routing
* Middleware
* PSR compliance
* Clean architecture

## Installation

```bash
composer require djalmamartins/organic-router
```

## Philosophy

Organic Router follows a minimalistic design inspired by modern frameworks while keeping the core extremely lightweight.

## Features (planned)

* HTTP routing (GET, POST, PUT, DELETE)
* Route parameters
* Subdomain routing
* Middleware support
* Route caching
* PSR-7 compatibility
* PSR-15 middleware support
